# Spectre-monitor-dashboard
### Spectre-monitor 后端数据处理模块

* 安装

切换至 webserver 目录:
```
cd /home/wwwroot
```
Git clone:
```shell
git clone https://github.com/xiyouant/Spectre-monitor-BE.git
```
覆盖 `default` 内的原始文件:
```shell
mv Spectre-monitor-BE/* default
```

* 启动

浏览器访问:

`homepage`:
```shell
192.168.2.1
```

`Api 接口`:
```shell
192.168.2.1/api/meta_json
```
